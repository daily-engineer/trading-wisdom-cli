# 统一框架架构设计

## 分层架构

```
┌─────────────────────────────────────┐
│     交易决策层                      │
│  - 投资组合管理                     │
│  - 风险管理                         │
│  - 订单执行                         │
└──────────────┬──────────────────────┘
               │
┌──────────────▼──────────────────────┐
│     AI分析层                        │
│  - 多代理分析（TradingAgents）      │
│  - 基本面/技术面/情感分析          │
│  - 期权专项分析                     │
└──────────────┬──────────────────────┘
               │
┌──────────────▼──────────────────────┐
│     策略执行层                      │
│  - VNpy策略引擎                     │
│  - 回测框架                         │
│  - 纸币交易                         │
└──────────────┬──────────────────────┘
               │
┌──────────────▼──────────────────────┐
│     市场数据层                      │
│  - Tushare（A股）                   │
│  - AkShare（期权）                  │
│  - IB API（港美股）                 │
└─────────────────────────────────────┘
```

## 核心特性

- **多代理AI系统** - LLM驱动的智能分析
- **专业量化执行** - VNpy专业级交易引擎
- **完整期权支持** - Greeks计算、风控、对冲
- **多市场统一** - A股/港股/美股无缝切换
- **风控系统** - 多维度风险管理和监控

详见项目架构文档。
